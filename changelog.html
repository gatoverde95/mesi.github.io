Changelog

Release history and bug reports

Track the evolution of CuerdOS through our detailed changelog and known issues.

Release History

Cessna 1.2.1

Latest
  • Base updated to Debian 12.9
  • Migration to CuerdOS own repositories
  • Brave web browser set as default, Firefox stable still available in repositories
  • CuerdOS Standard now features Sway Notification Center as new notification manager
  • CuerdOS Legacy now includes Clipman plugin for Xfce4 by default
  • Firmware updates coming soon via "apt update"
  • Select-Install deprecated due to discontinuation of NHS kernel package
  • Improved Calamares installer configuration, facilitating selection of different file systems without manual partitioning

Cessna 1.2

  • Waydroid support enabled
  • Kernel with patch to improve performance (BORE and Ananicy service)
  • Optimized system boot speed
  • Select-install installer, for kernel selection (and ISO Standard window manager)
  • New animation from Plymouth, CuerdOS-simple, returning to the origins where less is more
  • Improved theme consistency between QT programs in GTK environments and vice versa
  • The Kisak Mesa drivers with their repository are set by default
  • Default Nvidia driver installer in both versions (Legacy and Standard)
  • Preload deprecated to find other more consistent optimizations
  • Changes in the system software suite

Cessna 1.1

  • BTRFS file system support (already added in the latest CuerdOS 1.0 maintenance ISO)
  • BTRFS is now the default file system through the Calamares installer
  • Image updated with the latest updates
  • Xanmod kernel support dropped for new CuerdOS kernel, version 6.6.23
  • Preload is installed and configured by default
  • I3 and Sway configurations moved to separate packages
  • New wallpapers available, plus community wallpapers package
  • New packages available in the repository
  • Some programs pinned to taskbar in CuerdOS-Legacy
  • New Welcome screen with distribution information

Cessna 1.0

  • ISO updated to Debian 12.4
  • Changed LS and CAT to LSD and BAT for improved terminal experience
  • New system administration programs: cuerdos-control-panel, cups-switch and setup-repos
  • Changed to Xanmod LTS V1 kernel for better performance
  • Added retro games: Doom, Tetris and Pacman from DosBox
  • New color theme for Gedit
  • Updated Sway and I3 configurations
  • New Xfce edition called CuerdOS-Legacy
